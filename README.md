# ♟️ Pro-Chess Web App

A clean, responsive, and lightweight web-based Chess game built with JavaScript. This project is designed for seamless performance and ease of use, allowing you to play against a computer opponent directly in your browser.

## 🚀 Features
*   **Drag-and-Drop Interface:** Intuitive piece movement.
*   **AI Opponent:** Automatic random move generator for single-player practice.
*   **Responsive Design:** Works smoothly on desktop and mobile devices.
*   **Zero Dependencies:** Designed as a single-file application for fast loading and offline capabilities.
*   **GitHub Pages Ready:** Deploy instantly via GitHub.

## 🛠️ Tech Stack
*   **HTML5** & **CSS3**
*   **JavaScript (Vanilla)**
*   **Chess.js:** For game logic and move validation.
*   **Chessboard.js:** For the interactive board UI.
